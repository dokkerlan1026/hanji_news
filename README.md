# hanji_news

這個 repo 用來存放老爸交代我整理的日常資料，格式如下：

- 每天建立一個日期資料夾，例如 `2026-03-17/`
- 當天新增的內容以 Markdown 檔存放在該資料夾內
- 檔名格式為 `HHMM_中文標題.md`
- `README.md` 作為總目錄，記錄每天資料夾內的內容摘要、標籤與連結
- 如果某天沒有新增檔案，就跳過，不特別建立條目
