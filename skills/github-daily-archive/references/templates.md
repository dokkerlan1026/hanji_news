# Templates

## Filename template

```text
HHMM_中文標題.md
```

Examples:

- `0700_台股早報.md`
- `1115_任務整理.md`
- `2130_讀書筆記.md`

## README day section template

```md
## YYYY-MM-DD

資料夾：[`YYYY-MM-DD/`](./YYYY-MM-DD/)

- [`HHMM_標題.md`](./YYYY-MM-DD/<url-encoded-filename>)
  - 標籤：`標籤 1` `標籤 2`
  - 摘要：一句話摘要。
```

## Summary template

Use one sentence. Mention only the most useful retrieval clues:

- what it is
- time window or scope if relevant
- main topic or outcome

Examples:

- 整理今日 07:00 - 12:00 的台股早盤新聞，聚焦 AI、台積電與記憶體族群。
- 記錄今天的任務優先順序、待辦事項與後續行動。
- 摘要一篇文章的核心觀點，附上可直接採取的做法。
