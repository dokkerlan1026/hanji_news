# Prompt and response patterns

## Useful query prompts

- `幫我查這個網站在說什麼`
- `幫我找這頁的重點`
- `幫我比較這 3 個頁面的差異`
- `幫我看這個表單要怎麼填`
- `幫我打開頁面後點到某個區塊`

## Good response pattern for lookup

```text
目標：找出這頁在講什麼

結果：
- 重點 1
- 重點 2
- 重點 3

來源：
- https://example.com
```

## Good response pattern for interaction

```text
目前頁面：登入頁
已完成：
- 找到 Email 欄位
- 找到密碼欄位
- 找到登入按鈕

下一步：
- 等你確認是否要送出登入
```

## Good response pattern for extraction

```text
我抓到的資訊：
- 名稱：...
- 價格：...
- 狀態：...
```
