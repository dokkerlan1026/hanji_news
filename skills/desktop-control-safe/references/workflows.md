# Desktop workflows

## 1. Click a visible target

1. Identify the window.
2. Confirm the target area visually.
3. Move mouse.
4. Click once.
5. Verify the result.

## 2. Type into an app

1. Activate the correct window.
2. Confirm the text field is focused.
3. Type text.
4. Verify the typed content when possible.

## 3. Drag and drop

1. Confirm source and destination.
2. Move to source.
3. Press and drag.
4. Release at destination.
5. Verify the object moved as intended.

## 4. Screenshot-driven control

1. Capture current state.
2. Decide next small action.
3. Execute one action.
4. Capture or inspect again.
