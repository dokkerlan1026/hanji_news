# Safety rules

## Ask or pause before

- deleting files
- closing unsaved work
- submitting money / payments
- sending messages or emails
- changing account or security settings
- anything visually ambiguous and irreversible

## Safer defaults

- one click over double-click unless needed
- verify active window before typing
- use small steps instead of long scripted chains
- report intended action before major UI changes

## Good wording

- `我先確認現在是哪個視窗。`
- `我先看一下畫面，再決定下一步。`
- `這一步可能會真的送出內容，我先停在這裡等你確認。`
