# Song templates

## 1. Emotional Mandopop

```text
[Verse 1]
Set the scene and emotional problem.

[Pre-Chorus]
Raise the emotional tension.

[Chorus]
Deliver the main hook with the clearest emotional statement.

[Verse 2]
Add a sharper image or memory.

[Pre-Chorus]
Escalate.

[Chorus]
Repeat with slight variation.

[Bridge]
Change perspective or reveal the core truth.

[Final Chorus]
Biggest emotional payoff.
```

## 2. Youth / Summer / School vibe

```text
[Intro]
Short image or chant-like setup.

[Verse 1]
Paint a bright scene.

[Chorus]
Simple, shoutable hook.

[Verse 2]
Add movement, dreams, friendship, or first-love detail.

[Bridge]
A promise or turning point.

[Final Chorus]
Bigger, more open, more anthemic.
```

## 3. Dark synth / alt pop

```text
[Verse 1]
Sparse and visual.

[Pre-Chorus]
Unease grows.

[Chorus]
Short, sticky, hypnotic hook.

[Verse 2]
More tension and contrast.

[Bridge]
Drop to minimal instrumentation or spoken-feel lines.

[Final Chorus]
Return with stronger impact.
```

## 4. Fast rock / pop punk

```text
[Verse 1]
Immediate conflict.

[Chorus]
Fast payoff, strong slogan-like hook.

[Verse 2]
Raise stakes.

[Chorus]
Repeat.

[Bridge]
Half-time or breakdown feel.

[Final Chorus]
Big close.
```
